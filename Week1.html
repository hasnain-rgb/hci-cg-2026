Introduction

Memory allows us to store and retrieve information, which is essential for everyday activities, learning, and performing sequences of actions. For example, memory games challenge our ability to recall and add to sequences of objects or actions. Our memory system can be divided into three main types: Sensory Memory, Short-Term (Working) Memory, and Long-Term Memory.

Sensory Memory

Definition

Sensory memory acts as a temporary buffer for incoming stimuli received through the senses. Each sensory channel has its own memory:

  • Iconic memory: Visual information (~0.5 seconds)
  • Echoic memory: Auditory information
  • Haptic memory: Touch

New incoming information constantly overwrites these memories.

Example: Seeing a moving sparkler leaves a persistent visual image; hearing a sound allows you to locate its direction.

Reference: Cognitive Psychology Texts

Short-Term (Working) Memory

Definition

Short-term memory acts as a “scratch-pad” for temporary storage of information actively used for tasks like calculations or comprehension.

Example: Calculating 35 × 6 mentally by breaking it into smaller steps, or remembering the beginning of a sentence while reading.

  • Access Speed: ~70 ms
  • Duration: ~200 ms
  • Capacity: ~7 ± 2 items (Miller, 1956)

Reference: Cognitive Psychology Texts

Long-Term Memory

Definition

Long-term memory stores all knowledge, experiences, and procedural skills. It has a huge capacity and slower access time, but information is retained for long periods, from hours to years.

  • Access Speed: ~0.1 s
  • Capacity: Practically unlimited
  • Decay: Very slow

Reference: Cognitive Psychology Texts

Comparison of Memory Types

Feature Sensory Memory Short-Term Memory Long-Term Memory
Function Temporary buffer for senses Temporary storage & manipulation Permanent storage of knowledge
Duration ~0.5 seconds ~200 ms Minutes to years
Capacity Very large (all sensory input) Limited (~7 ± 2 items) Practically unlimited
Access Speed Immediate Fast (~70 ms) Slower (~0.1 s)
Decay / Forgetting Rapid Rapid Slow
Example Seeing a moving finger, hearing a sound Mental calculations, reading comprehension Remembering facts, skills, life events
